配置 Cursor 以帮助您编写和维护 Mintlify 文档。

## 前置要求

- 已安装 Cursor 编辑器

## 配置步骤

<Steps>
<Step title="在 Cursor 中打开文档目录">
  打开包含 `docs.json` 的文档仓库根目录。
</Step>

<Step title="安装 Mintlify skill">
  在集成终端中,运行:

  ```bash
  npx skills add https://mintlify.com/docs
  ```

  这将为 Cursor 提供 Mintlify 的组件参考、写作标准和工作流指导。
</Step>

<Step title="自定义项目说明">
  编辑项目根目录中的 `AGENTS.md`,添加项目特定的术语、风格偏好和内容边界。
</Step>

<Step title="开始使用 Cursor 写作">
  打开文件并使用 Cursor 的 AI 功能起草和编辑文档。
</Step>
</Steps>
