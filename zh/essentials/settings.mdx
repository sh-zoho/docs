每个 Mintlify 站点都通过项目根目录中的 `docs.json` 文件进行配置。更改会立即在本地预览中生效。

## 身份标识

设置站点的名称、主题和视觉标识。

```json docs.json
{
  "name": "Acme Docs",
  "theme": "mint",
  "favicon": "/favicon.svg",
  "logo": {
    "light": "/logo/light.svg",
    "dark": "/logo/dark.svg"
  }
}
```

<ResponseField name="name" type="string" required>
  您的项目名称。显示在浏览器标签标题和站点元数据中。
</ResponseField>

<ResponseField name="theme" type="string">
  视觉主题。选项:`mint`、`maple`、`willow`、`sequoia`、`prism`。
</ResponseField>

<ResponseField name="favicon" type="string">
  网站图标的路径。建议使用 SVG 以在各种尺寸下都清晰呈现。
</ResponseField>

<ResponseField name="logo" type="object">
  <Expandable title="Logo">
    <ResponseField name="light" type="string">浅色模式下显示的 Logo。</ResponseField>
    <ResponseField name="dark" type="string">深色模式下显示的 Logo。</ResponseField>
    <ResponseField name="href" type="string" default="/">Logo 链接到的 URL。</ResponseField>
  </Expandable>
</ResponseField>

## 颜色

```json docs.json
{
  "colors": {
    "primary": "#16A34A",
    "light": "#07C983",
    "dark": "#15803D"
  }
}
```

<ResponseField name="colors.primary" type="string" required>
  主要品牌色。在浅色模式下用于高亮、激活状态和强调色。
</ResponseField>

<ResponseField name="colors.light" type="string">
  深色模式下的主色。
</ResponseField>

<ResponseField name="colors.dark" type="string">
  用于按钮和重要的交互元素。
</ResponseField>

<ResponseField name="colors.background" type="object">
  覆盖默认页面背景颜色。
  <Expandable title="Background">
    <ResponseField name="light" type="string">浅色模式下的背景十六进制颜色。</ResponseField>
    <ResponseField name="dark" type="string">深色模式下的背景十六进制颜色。</ResponseField>
  </Expandable>
</ResponseField>

## 导航栏

配置顶部导航栏的链接和主要操作按钮。

```json docs.json
{
  "navbar": {
    "links": [
      { "label": "Support", "href": "mailto:support@example.com" }
    ],
    "primary": {
      "type": "button",
      "label": "Dashboard",
      "href": "https://dashboard.example.com"
    }
  }
}
```

<ResponseField name="navbar.links" type="array">
  顶部导航栏中显示的文本链接。
  <Expandable title="Link">
    <ResponseField name="label" type="string">链接的显示文本。</ResponseField>
    <ResponseField name="href" type="string">链接指向的 URL。</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="navbar.primary" type="object">
  右上角醒目的操作按钮。
  <Expandable title="Primary">
    <ResponseField name="type" type='"button" | "github"'>
      `button` 渲染为 CTA 按钮。`github` 渲染为星标数徽章。
    </ResponseField>
    <ResponseField name="label" type="string">按钮标签。当类型为 `button` 时必填。</ResponseField>
    <ResponseField name="href" type="string">按钮或徽章链接到的 URL。</ResponseField>
  </Expandable>
</ResponseField>

## 导航

将页面组织到标签页和分组中。完整详情请参阅[导航](/essentials/navigation)指南。

```json docs.json
{
  "navigation": {
    "tabs": [
      {
        "tab": "Guides",
        "groups": [
          {
            "group": "Getting started",
            "pages": ["index", "quickstart"]
          }
        ]
      }
    ],
    "global": {
      "anchors": [
        {
          "anchor": "Blog",
          "href": "https://example.com/blog",
          "icon": "newspaper"
        }
      ]
    }
  }
}
```

## 上下文工具栏

上下文工具栏浮动显示在每个页面上,让读者一键访问 AI 工具和集成。完整选项列表请参阅[上下文工具栏](/agent-ready/contextual)。

```json docs.json
{
  "contextual": {
    "options": ["copy", "view", "claude", "mcp", "cursor", "vscode"]
  }
}
```

## 页脚

```json docs.json
{
  "footer": {
    "socials": {
      "x": "https://x.com/yourhandle",
      "github": "https://github.com/yourorg",
      "linkedin": "https://linkedin.com/company/yourco"
    }
  }
}
```

<ResponseField name="footer.socials" type="object">
  页脚中显示的社交链接。支持的键:`x`、`github`、`linkedin`、`discord`、`slack`、`facebook`、`instagram`、`website`。
</ResponseField>
