Mintlify 组件让您能够使用 MDX 构建丰富的交互式文档页面。这里列出了一些常用的组件。

完整列表请参阅 Mintlify 文档中的[组件概述](https://mintlify.com/docs/components)。

## 提示框

使用提示框来吸引读者对重要信息的注意。

<Note>**Note** 用于向读者提供有帮助的一般信息。</Note>

<Tip>**Tip** 突出有用的快捷方式或最佳实践。</Tip>

<Warning>**Warning** 标记可能引起问题的事项。</Warning>

<Info>**Info** 提示框提供补充上下文。</Info>

```mdx
<Note>A **Note** is for general information that helps the reader.</Note>

<Tip>A **Tip** highlights a helpful shortcut or best practice.</Tip>

<Warning>A **Warning** flags something that could cause problems.</Warning>

<Info>An **Info** callout provides supplementary context.</Info>
```

## 卡片

卡片将读者链接到其他页面或资源。

<Columns cols={2}>
  <Card title="API Reference" icon="terminal" href="/api-reference/introduction">
    根据您的 OpenAPI 规范自动生成的文档。
  </Card>
  <Card title="Quickstart" icon="rocket" href="/quickstart">
    分三步让您的文档运行起来。
  </Card>
</Columns>

```mdx
<Columns cols={2}>
  <Card title="API Reference" icon="terminal" href="/api-reference/introduction">
    Auto-generated docs from your OpenAPI spec.
  </Card>
  <Card title="Quickstart" icon="rocket" href="/quickstart">
    Get your docs running in three steps.
  </Card>
</Columns>
```

使用 `<Columns cols={N}>` 来控制网格宽度。没有 `href` 的卡片会渲染为信息面板。

## 步骤

步骤清晰地呈现编号的过程。

<Steps>
  <Step title="安装包">
    ```bash
    npm install my-package
    ```
  </Step>
  <Step title="初始化">
    在项目根目录运行 init 命令。
    ```bash
    npx my-package init
    ```
  </Step>
  <Step title="准备就绪">
    开始在您的代码中使用该包。
  </Step>
</Steps>

````mdx
<Steps>
  <Step title="Install the package">
    ```bash
    npm install my-package
    ```
  </Step>
  <Step title="Initialize">
    Run the init command in your project root.
  </Step>
  <Step title="You're ready">
    Start using the package in your code.
  </Step>
</Steps>
````

## 选项卡

选项卡让读者可以在替代内容之间切换——不同的语言、环境或平台。

<Tabs>
  <Tab title="npm">
    ```bash
    npm install my-package
    ```
  </Tab>
  <Tab title="yarn">
    ```bash
    yarn add my-package
    ```
  </Tab>
  <Tab title="pnpm">
    ```bash
    pnpm add my-package
    ```
  </Tab>
</Tabs>

````mdx
<Tabs>
  <Tab title="npm">
    ```bash
    npm install my-package
    ```
  </Tab>
  <Tab title="yarn">
    ```bash
    yarn add my-package
    ```
  </Tab>
  <Tab title="pnpm">
    ```bash
    pnpm add my-package
    ```
  </Tab>
</Tabs>
````

## 手风琴

手风琴将内容折叠在切换按钮后面——适用于常见问题或可选详细信息。

<AccordionGroup>
  <Accordion title="这个产品有什么用?">
    对第一个问题的简短回答。保持简洁。
  </Accordion>
  <Accordion title="价格是如何计算的?">
    另一个答案。根据需要使用尽可能多的空间。
  </Accordion>
</AccordionGroup>

```mdx
<AccordionGroup>
  <Accordion title="What does this product do?">
    A brief answer to the first question.
  </Accordion>
  <Accordion title="How is pricing calculated?">
    Another answer here.
  </Accordion>
</AccordionGroup>
```

## 框架

将图像或其他内容包装在 `Frame` 中,以添加带样式的边框和圆角。

<Frame>
  <img src="https://mintlify-assets.b-cdn.net/bigbend.jpg" alt="Big Bend 风景" />
</Frame>

```mdx
<Frame>
  <img src="/images/screenshot.png" alt="App screenshot" />
</Frame>
```

## 代码组

`CodeGroup` 将多个代码片段显示为选项卡——非常适合以多种语言显示相同的示例。

<CodeGroup>
  ```typescript TypeScript
  const response = await fetch('https://api.example.com/users');
  const users = await response.json();
  ```

  ```python Python
  import requests
  response = requests.get('https://api.example.com/users')
  users = response.json()
  ```

  ```bash curl
  curl https://api.example.com/users
  ```
</CodeGroup>

````mdx
<CodeGroup>
  ```typescript TypeScript
  const response = await fetch('https://api.example.com/users');
  ```

  ```python Python
  response = requests.get('https://api.example.com/users')
  ```
</CodeGroup>
````
