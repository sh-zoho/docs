Mintlify 组件让您能够使用 MDX 构建丰富的交互式文档页面。以下是一些常用组件。

完整列表请参阅 Mintlify 文档中的[组件概览](https://mintlify.com/docs/components)。

## 提示框

使用提示框来引起对重要信息的关注。

<Note>**Note**(注意)用于提供对读者有帮助的一般性信息。</Note>

<Tip>**Tip**(提示)突出显示有用的快捷方式或最佳实践。</Tip>

<Warning>**Warning**(警告)标记可能导致问题的事项。</Warning>

<Info>**Info**(信息)提示框提供补充上下文。</Info>

```mdx
<Note>A **Note** is for general information that helps the reader.</Note>

<Tip>A **Tip** highlights a helpful shortcut or best practice.</Tip>

<Warning>A **Warning** flags something that could cause problems.</Warning>

<Info>An **Info** callout provides supplementary context.</Info>
```

## 卡片

卡片将读者链接到其他页面或资源。

<Columns cols={2}>
  <Card title="API Reference" icon="terminal" href="/api-reference/introduction">
    根据您的 OpenAPI 规范自动生成的文档。
  </Card>
  <Card title="Quickstart" icon="rocket" href="/quickstart">
    通过三个步骤让您的文档运行起来。
  </Card>
</Columns>

```mdx
<Columns cols={2}>
  <Card title="API Reference" icon="terminal" href="/api-reference/introduction">
    Auto-generated docs from your OpenAPI spec.
  </Card>
  <Card title="Quickstart" icon="rocket" href="/quickstart">
    Get your docs running in three steps.
  </Card>
</Columns>
```

使用 `<Columns cols={N}>` 来控制网格宽度。没有 `href` 的卡片会渲染为信息面板。

## 步骤

步骤组件清晰地呈现编号的流程。

<Steps>
  <Step title="安装软件包">
    ```bash
    npm install my-package
    ```
  </Step>
  <Step title="初始化">
    在项目根目录运行初始化命令。
    ```bash
    npx my-package init
    ```
  </Step>
  <Step title="准备就绪">
    开始在代码中使用该软件包。
  </Step>
</Steps>

````mdx
<Steps>
  <Step title="Install the package">
    ```bash
    npm install my-package
    ```
  </Step>
  <Step title="Initialize">
    Run the init command in your project root.
  </Step>
  <Step title="You're ready">
    Start using the package in your code.
  </Step>
</Steps>
````

## 标签页

标签页让读者在不同内容之间切换——不同的语言、环境或平台。

<Tabs>
  <Tab title="npm">
    ```bash
    npm install my-package
    ```
  </Tab>
  <Tab title="yarn">
    ```bash
    yarn add my-package
    ```
  </Tab>
  <Tab title="pnpm">
    ```bash
    pnpm add my-package
    ```
  </Tab>
</Tabs>

````mdx
<Tabs>
  <Tab title="npm">
    ```bash
    npm install my-package
    ```
  </Tab>
  <Tab title="yarn">
    ```bash
    yarn add my-package
    ```
  </Tab>
  <Tab title="pnpm">
    ```bash
    pnpm add my-package
    ```
  </Tab>
</Tabs>
````

## 手风琴

手风琴将内容折叠在一个切换开关后面——适用于常见问题或可选详情。

<AccordionGroup>
  <Accordion title="这个产品是做什么的?">
    对第一个问题的简要回答。保持简洁。
  </Accordion>
  <Accordion title="价格是如何计算的?">
    这里是另一个答案。可以根据需要使用更多空间。
  </Accordion>
</AccordionGroup>

```mdx
<AccordionGroup>
  <Accordion title="What does this product do?">
    A brief answer to the first question.
  </Accordion>
  <Accordion title="How is pricing calculated?">
    Another answer here.
  </Accordion>
</AccordionGroup>
```

## 边框

将图片或其他内容包装在 `Frame` 中以添加带样式的边框和圆角。

<Frame>
  <img src="https://mintlify-assets.b-cdn.net/bigbend.jpg" alt="Big Bend landscape" />
</Frame>

```mdx
<Frame>
  <img src="/images/screenshot.png" alt="App screenshot" />
</Frame>
```

## 代码分组

`CodeGroup` 将多个代码片段显示为标签页——非常适合以多种语言展示相同的示例。

<CodeGroup>
  ```typescript TypeScript
  const response = await fetch('https://api.example.com/users');
  const users = await response.json();
  ```

  ```python Python
  import requests
  response = requests.get('https://api.example.com/users')
  users = response.json()
  ```

  ```bash curl
  curl https://api.example.com/users
  ```
</CodeGroup>

````mdx
<CodeGroup>
  ```typescript TypeScript
  const response = await fetch('https://api.example.com/users');
  ```

  ```python Python
  response = requests.get('https://api.example.com/users')
  ```
</CodeGroup>
````
