## 图片

### 使用 Markdown

[Markdown 语法](https://www.markdownguide.org/basic-syntax/#images)可以让您使用以下代码添加图片

```md
![title](/path/image.jpg)
```

请注意,图片文件大小必须小于 5MB。否则,我们建议将其托管在 [Cloudinary](https://cloudinary.com/) 或 [S3](https://aws.amazon.com/s3/) 等服务上。然后您可以使用该 URL 进行嵌入。

### 使用嵌入

要获得更多图片自定义功能,您还可以使用[嵌入](/writing-content/embed)来添加图片

```html
<img height="200" src="/path/image.jpg" />
```

## 嵌入和 HTML 元素

<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube video player"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
  style={{ width: '100%', borderRadius: '0.5rem' }}
></iframe>

<br />

<Tip>

Mintlify 支持 [Markdown 中的 HTML 标签](https://www.markdownguide.org/basic-syntax/#html)。如果您更喜欢 HTML 标签而非 Markdown 语法,这将很有帮助,让您能够以无限的灵活性创建文档。

</Tip>

### iFrames

在文档中加载另一个 HTML 页面。最常用于嵌入视频。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
